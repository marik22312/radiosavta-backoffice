## How to run locally

1. Set up required env variables
```env
REACT_APP_BASE_API_URL=""
REACT_APP_BASE_IMAGES_URL=""
REACT_APP_LIVE_STREAM_URL=""
REACT_APP_BASE_CLIENT_URL=""
```
2. Install dependencies running 
```bash
yarn install
```
3. Run the application
```bash
yarn start
```
